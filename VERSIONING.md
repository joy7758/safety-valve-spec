# Versioning (SVS)

## Semantic Versioning (adapted)
- v0.x: additive changes only; no breaking field semantic changes.
- v1.0: core semantics frozen; breaking changes require major version bump.

## Canonical Versions
- Receipt Spec: `spec/receipt-v0.1.md`
- Schema: `schemas/receipt.v0.1.json`
- Tests: `conformance/tests.md`
